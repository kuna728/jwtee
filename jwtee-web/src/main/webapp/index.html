Client-side session library for Java EE

JWTee is a library for the Java EE platform that provides a client-side user session mechanism. The session is implemented in the form of JWT tokens. The library uses the well-known HttpSession interface to manage the session. The library is fully configurable, however, no configuration is required for it to work. Documentation in the form of javadoc and sample applications are available under the tabs above.
